# Betrock
 A clean-room implementation of b1.7.3 in C++
 Made to be as performant as possible!

# Goals
- Be fully compatible with existent b1.7.3 Servers, Clients, Maps and Resource Packs
- Be easy to understand and modify
- Be near indistinguishable from the original b1.7.3

# Potential Goals
- Modding Support
	- Lua API
	- Shaders
		- Full access to the rendering pipeline
	- 3D Models through Resource Packs
